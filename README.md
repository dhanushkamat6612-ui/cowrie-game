**Authentic 5×5 Chowkabara with REAL-TIME KAUDE (Cowrie Shells), 2-4 Players, Spiral Movement, Joda Rules & All Traditional Rules!**

## ✨ Features
- 🎮 **2-4 Players** with custom names & unique colors
- 🐚 **Real-time 4 Kaude** animation (0/4 mouths = 8/4 points)
- 🌀 **Spiral Ring Movement** (anticlockwise → clockwise)
- ⚔️ **Complete Rules**: Joda, Triplet Invalid, Inner Ring Entry, Safe Squares
- 🎨 **Kamat Studio Boot Logo** + Premium Indian Aesthetics
- 📱 **Fully Responsive** Mobile/Desktop
- 🔔 **Rule Enforcement Popups** for all special cases

## 🎮 How to Play
1. **Boot Logo** → **Player Selection** → **Game Start**
2. **Roll 4 Kaude** above screen (rotate animation)
3. **Select your pawn** → **Move to valid square**
4. **Capture opponents**, make **Joda** in inner ring
5. **First to center with all pawns WINS!**

## 🐚 Kaude Values

Shells                                  Value                                  Effect

0 mouths                                8 score                                Extra throw + 8 steps

1 mouth                                 1 score                                1 step 

2 mouths                                2 score                                2 steps

3 mouths                                3 score                                3 steps

4 mouths                                4 score                                Extra throw + 4 steps
